System status
  • Healthy Latest poll succeeded and every device responded.
  • Setup needed No BLE transport configured yet — visit Settings → Setup and click "Find my dongle".
  • BLE not connected Transport is configured but the dongle isn't reachable. Check the BT-2 is powered + in range; on Docker, check the passthrough.
  • Comms slow No successful poll in 2+ minutes — usually a brief BLE wobble; clears on the next cycle.
  • X errors Last poll completed but some devices didn't reply. Common after a daemon restart (BlueZ holds the BT-2 connection for ~60s).
  • Stale No successful poll for 5+ minutes — check Bluetooth power / range to the dongle.
  • Offline Scheduler isn't running. Visit Settings → System → Restart daemon.

System

Restart the daemon to apply pending changes (new transports, devices, exporters). The UI reconnects automatically — takes ~5 seconds.

Web password

The local password protects writes from the LAN. Rotate it if you've forgotten yours, or if you suspect it's been shared. The new password is shown to you once — save it right then.

Kiosk share URL

Public chrome-free wall-display view of SoC + flow. Anyone with this URL can view live state (no login). Designed for wall-mounted tablets, Fire TV, digital signage. Rotate to instantly revoke a URL that's been over-shared.
